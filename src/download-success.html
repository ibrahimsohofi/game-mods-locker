Installation Instructions

  1. Download and install Minecraft Forge for version 1.16.5 through 1.20.1
  2. Extract the downloaded mod pack ZIP file
  3. Copy all mod files (.jar) to your Minecraft mods folder:
    • Windows: %appdata%\.minecraft\mods
    • Mac: ~/Library/Application Support/minecraft/mods
    • Linux: ~/.minecraft/mods
  4. Launch Minecraft with the Forge profile
  5. Enjoy your premium mods!
